# Ceeder

University project — a multi-page website for Ceeder, a smart automated pet feeder product. Built with semantic HTML5 and CSS3, with a server-side contact form using ASP.NET Web Forms.

## Features

- Responsive landing page with hero section and product showcase
- About and gallery pages
- Contact form with server-side processing (ASP.NET)
- Semantic HTML5 layout with custom CSS

## Getting Started

### Static pages (HTML/CSS)

Open `index.html` directly in a browser.

### Contact form (ASP.NET)

Requires IIS or Visual Studio local server:

1. Open the project in Visual Studio
2. Press F5 to run with IIS Express
3. Navigate to `contact.aspx`
